You can download *CloudMix* from the "FuseForge Maven Repository":http://repo.fusesource.com/maven2/ in the "CloudMix area":http://repo.fusesource.com/maven2/org/fusesource/cloudmix/.

If you use Maven then you can add the following to your *pom.xml*
 
<% coderay :lang => 'xml' do -%>
<dependency>
  <groupId>org.fusesource.cloudmix</groupId>
  <artifactId>cloudmix-core</artifactId>
  <version>${cloudmix-version}</version>  
</dependency>
<% end -%>


h2. Getting the source code

If you prefer you can try building from "the source code":source.html and "building it":building.html
